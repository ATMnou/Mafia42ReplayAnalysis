# Mafia42ReplayAnalysis

마피아42 리플레이 URL을 입력하면 채팅 로그와 유저 정보를 수집하고, OpenAI 호환 API를 이용해 한국어로 게임 흐름을 분석하는 Node.js 도구입니다.

## 주요 기능

- 마피아42 리플레이 페이지에서 대화 로그 수집
- 시스템 메시지, 일반 채팅, 메가폰 채팅 구분
- 유저 테이블에서 닉네임과 직업 아이콘 추출
- 수집한 데이터를 기반으로 AI 분석 요약 생성
- 한국어 결과 출력

## 준비 사항 (확장 프로그램 버전)

- Chrome 또는 Chromium 기반 브라우저
- OpenAI 호환 API 키

## 준비 사항 (CLI 버전)

- Node.js 18 이상 권장
- OpenAI 호환 API 키
- 분석 대상 리플레이 URL

## 설치

```bash
npm install
```

## 환경 변수 설정

프로젝트 루트에 `.env` 파일을 만들고 아래 값을 설정하세요.

```env
OPENAI_API_KEY=your_api_key_here
```

`.env.example` 파일을 참고해도 됩니다.

## 실행 방법

`main.js`에는 현재 분석할 리플레이 URL이 하드코딩되어 있습니다. 분석할 대상을 바꾸려면 `main.js`의 `mafiaUrl` 값을 수정하세요.

```bash
node main.js
```

실행 전에 같은 폴더에 `context.txt` 파일이 있어야 합니다. 이 파일의 내용도 함께 AI 분석 입력으로 사용됩니다.

## 동작 방식

1. 리플레이 URL에서 언어 코드와 리플레이 ID를 추출합니다.
2. 마피아42 공개 API 엔드포인트를 통해 HTML 데이터를 가져옵니다.
3. 채팅 로그와 유저 테이블 정보를 파싱합니다.
4. `context.txt`와 수집 로그를 합쳐 AI 모델에 전달합니다.
5. 한국어 분석 결과를 콘솔에 출력합니다.

## 출력 예시

- `[SYSTEM]` 형식의 시스템 메시지
- `[CHAT]`, `[MEGAPHONE]` 형식의 채팅 로그
- `[USER_TABLE]` 형식의 유저 목록
- 최종 AI 분석 요약

## 주의 사항

- 리플레이 URL 형식이 맞지 않으면 수집이 실패할 수 있습니다.
- 현재 코드는 `kr`, `en` 언어 리플레이만 처리합니다.
- OpenAI 호환 API를 사용하므로, 사용 중인 서비스의 모델명과 요금 정책을 확인하는 것이 좋습니다.
- 게임 정보가 매우 길어 입력 토큰이 1만을 쉽게 넘기므로, 모델 비용에 유의하세요.
- 이 프로젝트는 마피아42에서 만들거나 승인하지 않았습니다.
- 마피아.com에서 제공하는 리플레이 AI 분석 도구의 리버스 엔지니어링이 아니며, 공개 API와 웹 크롤링을 활용한 독립적인 구현입니다.
