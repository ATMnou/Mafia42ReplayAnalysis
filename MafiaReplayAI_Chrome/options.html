키 발급받기

키는 chrome.storage.sync에 저장되며, 브라우저 외부로 전송되지 않습니다. API 요금은 OpenRouter 정책을 따릅니다.

무료 모델은 OpenRouter 정책에 따라 하루 50회, 10달러 이상 충전 시 하루 1,000회까지 사용 가능합니다.

교주 게임 자동 감지: 참가자 중 교주(cultleader)가 있으면 확장된 컨텍스트 파일(context_cult.txt)이 자동으로 사용됩니다.
