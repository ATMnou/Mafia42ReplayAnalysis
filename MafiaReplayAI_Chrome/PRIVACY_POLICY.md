# 개인정보처리방침 (Mafia42 Replay Analyzer)

시행일: 2026-05-25

Mafia42 Replay Analyzer(이하 "확장 프로그램")는 사용자의 개인정보를 최소한으로 처리하며, 아래와 같이 처리 목적과 범위를 안내합니다.

## 1. 수집 및 처리하는 정보

확장 프로그램은 다음 정보를 처리할 수 있습니다.

1. 인증 정보

- OpenRouter API 키
- 저장 위치: `chrome.storage.sync`
- 처리 목적: AI 분석 요청 인증

2. 웹사이트 콘텐츠

- 마피아42 리플레이 페이지의 채팅 로그, 시스템 로그, 닉네임, 직업, 승리팀 등
- 처리 목적: 리플레이 분석 결과 생성

3. 웹 기록(제한적)

- 현재 분석 대상 페이지 URL(예: `/history/{lang}/{roomId}`)
- 처리 목적: 리플레이 데이터 조회 및 분석 요청 구성

## 2. 처리 목적

확장 프로그램은 아래 목적에 한해 정보를 처리합니다.

- 사용자가 요청한 리플레이 분석 제공
- 사용자가 저장한 설정(API 키, 모델 선택) 유지
- 분석 기능 동작에 필요한 외부 API 호출

## 3. 제3자 제공 및 외부 전송

확장 프로그램은 기능 수행을 위해 아래 서비스로 데이터를 전송할 수 있습니다.

1. OpenRouter (`https://openrouter.ai/*`)

- 전송 항목: 분석 대상 텍스트(채팅/로그/사용자 테이블 요약), API 키(인증 헤더)
- 목적: AI 분석 결과 생성

2. 리플레이 조회용 AWS 엔드포인트

- 예: `execute-api.ap-northeast-2.amazonaws.com`, `lambda-url.us-east-1.on.aws` 등
- 전송 항목: 리플레이 식별자(roomId), 언어(lang)
- 목적: 리플레이 원본 HTML 조회

확장 프로그램은 위 목적 외에 개인정보를 판매하지 않습니다.

## 4. 저장 및 보관 기간

- OpenRouter API 키 및 선택 모델은 사용자가 삭제하거나 확장 프로그램을 제거할 때까지 보관될 수 있습니다.
- 분석용 로그/콘텐츠는 확장 프로그램 내부에 별도 장기 저장하지 않습니다.
- 단, 외부 서비스(OpenRouter/AWS)의 로그 보관 정책은 각 서비스 약관 및 정책을 따릅니다.

## 5. 사용자 선택권

사용자는 다음 방법으로 정보 처리를 통제할 수 있습니다.

- 확장 프로그램 옵션 페이지에서 API 키 삭제
- 확장 프로그램 제거
- 브라우저의 확장 프로그램 데이터 삭제 기능 사용

## 6. 보안

확장 프로그램은 기능 수행에 필요한 최소 권한만 사용하도록 설계되었습니다.

- `storage`: 설정(API 키, 모델) 저장
- `host_permissions`: `mafia42.com`, 리플레이 조회 엔드포인트, OpenRouter 호출

## 7. 아동의 개인정보

확장 프로그램은 아동을 대상으로 특별히 설계되지 않았으며, 관련 법령을 준수합니다.

## 8. 정책 변경

본 개인정보처리방침은 기능 변경 또는 법령/정책 변경에 따라 수정될 수 있습니다. 중요한 변경 시 문서의 시행일을 업데이트합니다.
